# Color Game

A browser-based color matching game. You're shown a target color and must recreate it using HSV sliders (hue, saturation, value). The closer your guess, the higher your score.

## How to play

Open `index.html` in a browser. Drag the sliders until your swatch matches the target, then submit to see how close you got.

## Colorblind mode

The game is configured in colorblind mode as a joke, but the code can easily be tweaked to run normally.
